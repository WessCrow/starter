# CONTEXT — NutriLeve

## O que é
Landing page de consultoria de nutrição online. Objetivo: apresentar o
serviço e converter visitantes em conversas no WhatsApp.

## Para quem
Pessoas que querem reeducação alimentar leve, sem dietas restritivas.

## Estado atual
- Fase 1 — Foundation: estrutura react-vite criada, landing inicial no ar (local).
- Seções: Hero, Benefícios, Como Funciona, CTA final, Footer.
- CTA: link direto wa.me (sem backend).

## Stack (informativo)
React + Vite + TypeScript + Tailwind + pnpm. Alias `@/` → `src/`.

## Fora do escopo (v1)
Blog, pagamento online.

## Próximo passo
Rodar QA Gate e validar a página no navegador.
